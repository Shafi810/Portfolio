# Shafi Usmani | Frontend Developer Portfolio

🚀 A modern, high-performance frontend portfolio built with **React (Vite)**, **Tailwind CSS v4**, and **Framer Motion**.  
It showcases my skills, projects, and contact form integration with **EmailJS**.

---

## ✨ Features
- Responsive design with Tailwind CSS v4
- Smooth animations using Framer Motion
- Contact form powered by EmailJS
- SEO-friendly metadata and Open Graph tags
- Dark mode aesthetic

---

## 🛠️ Tech Stack
- **React (Vite)**
- **Tailwind CSS v4**
- **Framer Motion**
- **EmailJS**
- **GitHub Pages / Vercel Deployment**
